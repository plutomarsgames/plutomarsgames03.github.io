# PolyTrack Racing Game

## Files Included
- index.html
- style.css
- game.js

## How To Upload

1. Create a GitHub repository.
2. Upload all files.
3. Go to:
   Settings -> Pages
4. Select:
   Deploy from branch -> main
5. Save.

Your website will become live.

## Where To Change Things

### Change Game Name
Open:
index.html

Change:
POLYTRACK

### Change Car Color
Open:
game.js

Find:
#00ffcc

Replace with any color.

### Change Background
Open:
style.css

Find:
background: #111;

### Change Speed
Open:
game.js

Find:
maxSpeed: 10

Increase number for faster car.
